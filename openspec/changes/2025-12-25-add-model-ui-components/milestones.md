# Milestones

## M1 - Requirements & Acceptance
- Entry criteria: Goal defined and scope confirmed.
- Exit criteria: Requirements + acceptance criteria drafted.
- Required artifacts: `requirements-analysis.md`, `acceptance-criteria.md`.

## M2 - OpenSpec Proposal
- Entry criteria: M1 completed.
- Exit criteria: Proposal + spec delta + tasks drafted.
- Required artifacts: `proposal.md`, `specs/vibescore-tracker/spec.md`, `tasks.md`.

## M3 - UI Component Implementation
- Entry criteria: Proposal approved.
- Exit criteria: Components added and copy registry updated.
- Required artifacts: Component files + copy registry entries.

## M4 - Manual Verification
- Entry criteria: Components implemented.
- Exit criteria: Manual render verification performed.
- Required artifacts: Verification notes.

## M5 - Release
- Entry criteria: M4 completed.
- Exit criteria: Changes merged and docs updated.
- Required artifacts: Updated docs and changelog (if applicable).
