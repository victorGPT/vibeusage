# Milestones

## M1 - Requirements & Acceptance
- Entry criteria: 目标与无滞后约束确认。
- Exit criteria: 需求与验收标准完成。
- Required artifacts: `requirements-analysis.md`, `acceptance-criteria.md`.

## M2 - OpenSpec Proposal
- Entry criteria: M1 完成。
- Exit criteria: 提案、任务清单、规格增量完成。
- Required artifacts: `proposal.md`, `tasks.md`, `specs/vibescore-tracker/spec.md`.

## M3 - Implementation
- Entry criteria: 提案获批。
- Exit criteria: DB RPC、Edge Function、测试与文档完成。
- Required artifacts: SQL 文件、Edge 更新、测试、`insforge-functions/` 构建产物。

## M4 - Verification
- Entry criteria: M3 完成。
- Exit criteria: 测试通过、回归声明记录、验证报告完成。
- Required artifacts: `verification-report.md`.
