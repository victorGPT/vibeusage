# Milestones

## M1 - Requirements & Acceptance
- Entry criteria: 需求确认（A/B/C）
- Exit criteria: `requirements-analysis.md` + `acceptance-criteria.md` 完成
- Required artifacts: requirements + acceptance

## M2 - OpenSpec Proposal (if applicable)
- Entry criteria: M1 完成
- Exit criteria: `proposal.md` + `tasks.md` + spec delta 完成
- Required artifacts: OpenSpec change set

## M3 - Unit Test Coverage
- Entry criteria: M2 完成
- Exit criteria: 并发守卫单元/合成测试完成
- Required artifacts: acceptance script or unit test

## M4 - Regression & Integration
- Entry criteria: M3 完成
- Exit criteria: ingest 回归路径验证完成
- Required artifacts: 运行命令与结果记录

## M5 - Release & Monitoring
- Entry criteria: M4 完成
- Exit criteria: canary 可运行 + 日志可观测
- Required artifacts: canary 脚本与运行说明
