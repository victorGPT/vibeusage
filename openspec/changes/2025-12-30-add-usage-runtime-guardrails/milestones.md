# Milestones

## M1 - Requirements & Acceptance
- Entry criteria: Scope confirmed; assumptions recorded.
- Exit criteria: Requirements + acceptance criteria authored.
- Required artifacts: `requirements-analysis.md`, `acceptance-criteria.md`.

## M2 - OpenSpec Proposal (if applicable)
- Entry criteria: M1 complete.
- Exit criteria: Proposal + tasks + spec deltas drafted and validated.
- Required artifacts: `proposal.md`, `tasks.md`, `specs/vibescore-tracker/spec.md`.

## M3 - Unit Test Coverage
- Entry criteria: Proposal approved.
- Exit criteria: Unit tests for range validation and logging threshold.
- Required artifacts: Test updates in `test/`.

## M4 - Regression & Integration
- Entry criteria: M3 complete.
- Exit criteria: Integration/acceptance steps executed; regression statement recorded.
- Required artifacts: test logs or scripts, verification notes.

## M5 - Release & Monitoring
- Entry criteria: M4 complete.
- Exit criteria: Functions deployed; logs confirm slow-query visibility.
- Required artifacts: deployment evidence, monitoring notes.
