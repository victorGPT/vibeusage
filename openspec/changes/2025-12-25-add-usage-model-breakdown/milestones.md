# Milestones

## M1 - Requirements & Acceptance
- Entry criteria: change ID created.
- Exit criteria: requirements analysis + acceptance criteria drafted.
- Required artifacts: `requirements-analysis.md`, `acceptance-criteria.md`.

## M2 - OpenSpec Proposal (if applicable)
- Entry criteria: M1 complete.
- Exit criteria: proposal + spec delta validated.
- Required artifacts: `proposal.md`, `specs/vibescore-tracker/spec.md`, `tasks.md`, `openspec validate --strict`.

## M3 - Unit Test Coverage
- Entry criteria: endpoint design approved.
- Exit criteria: grouping/cost helper tests added or documented as N/A with rationale.
- Required artifacts: test file or documented runbook.

## M4 - Regression & Integration
- Entry criteria: endpoint implemented.
- Exit criteria: integration/regression verification executed with evidence.
- Required artifacts: verification report updates + logs.

## M5 - Release & Monitoring
- Entry criteria: M4 complete.
- Exit criteria: function deployed and docs updated.
- Required artifacts: deployment notes, updated `BACKEND_API.md`.
