# Milestones

## M1 - Requirements & Acceptance
- Entry criteria: Scope confirmed; assumptions recorded.
- Exit criteria: Requirements + acceptance criteria authored.
- Required artifacts: `requirements-analysis.md`, `acceptance-criteria.md`.

## M2 - OpenSpec Proposal
- Entry criteria: M1 complete.
- Exit criteria: Proposal + tasks + spec deltas drafted.
- Required artifacts: `proposal.md`, `tasks.md`, `specs/vibescore-tracker/spec.md`.

## M3 - Implementation
- Entry criteria: Proposal approved.
- Exit criteria: Default updated, docs/tests updated, functions rebuilt.
- Required artifacts: code changes, updated docs, build artifacts.

## M4 - Verification
- Entry criteria: M3 complete.
- Exit criteria: Tests + acceptance checks complete; regression statement recorded.
- Required artifacts: test logs, verification notes.
