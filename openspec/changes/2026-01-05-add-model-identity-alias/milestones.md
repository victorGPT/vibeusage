# Milestones

## M1 - Requirements & Acceptance
- Entry criteria: change ID created.
- Exit criteria: requirements analysis + acceptance criteria drafted.
- Required artifacts: `requirements-analysis.md`, `acceptance-criteria.md`.

## M2 - OpenSpec Proposal
- Entry criteria: M1 complete.
- Exit criteria: proposal + spec delta validated.
- Required artifacts: `proposal.md`, `specs/vibescore-tracker/spec.md`, `tasks.md`, `openspec validate --strict`.

## M3 - Module Brief
- Entry criteria: M2 complete.
- Exit criteria: module brief approved.
- Required artifacts: `module-brief.md`.

## M4 - Implementation + Unit Tests
- Entry criteria: module brief approved.
- Exit criteria: shared resolver + API updates + unit tests green.
- Required artifacts: updated edge functions + test files.

## M5 - Regression & Verification
- Entry criteria: M4 complete.
- Exit criteria: regression run and verification report recorded.
- Required artifacts: verification notes + test logs.
