# Milestones

## M1 - Requirements & Acceptance
- Entry criteria: Problem statement confirmed; scope locked.
- Exit criteria: Requirement analysis + acceptance criteria approved.
- Required artifacts: `requirements-analysis.md`, `acceptance-criteria.md`.

## M2 - OpenSpec Proposal
- Entry criteria: M1 complete.
- Exit criteria: Proposal + tasks + spec deltas created; `openspec validate` clean.
- Required artifacts: `proposal.md`, `tasks.md`, `specs/vibescore-tracker/spec.md`.

## M3 - Implementation & Tests
- Entry criteria: Proposal approved.
- Exit criteria: SQL migration, ingest metrics write, retention update, acceptance tests added.
- Required artifacts: SQL files, updated functions, acceptance script.

## M4 - Verification & Rollout
- Entry criteria: M3 complete.
- Exit criteria: Verification queries run and recorded; deploy completed.
- Required artifacts: verification notes, deployment evidence.
