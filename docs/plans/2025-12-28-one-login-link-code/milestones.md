# Milestones

## M1 - Requirements & Acceptance
- Entry criteria: Design direction confirmed.
- Exit criteria: `requirements-analysis.md` + `acceptance-criteria.md` approved.
- Required artifacts: Requirements + Acceptance docs.

## M2 - OpenSpec Proposal (if applicable)
- Entry criteria: M1 complete.
- Exit criteria: OpenSpec change scaffolded with proposal/tasks/spec delta.
- Required artifacts: `openspec/changes/<id>/proposal.md`, `tasks.md`, `specs/**`.

## M3 - Unit Test Coverage
- Entry criteria: Implementation ready to test.
- Exit criteria: UI masking/copy tests and helper unit tests passing.
- Required artifacts: Test files + results.

## M4 - Regression & Integration
- Entry criteria: M3 complete.
- Exit criteria: Exchange/expiry/idempotency tests pass; CLI init regression verified.
- Required artifacts: Integration tests or acceptance scripts + results.

## M5 - Release & Monitoring
- Entry criteria: M4 complete.
- Exit criteria: Deployment complete with verification evidence.
- Required artifacts: PR regression statement + verification report.
